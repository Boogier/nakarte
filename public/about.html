About "iOrient - competition map"

This site is developed by Alexey Kozlov aka Boogier, a rogainer, trail runner and nature lover.

The site can display a map of a competition registered in iOrient.ru portal with geopositioning data provided by competing teams.

Open Source

This site is open source software built on base of Nakarte. The source code is available on GitHub.
